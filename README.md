## 📌 프로젝트 개요
본 프로젝트는 서울시 상권 데이터를 활용하여  
**서울시 5대 권역 간 소비 항목 비중의 차이가 통계적으로 유의미한지**를 분석하는 것을 목표로 한다.

단순 매출 규모 비교가 아닌,  
**총지출 대비 소비 항목 비중**을 기준으로 지역 간 소비 구조의 차이를 탐색하였으며,  
탐색적 데이터 분석(EDA)과 분산분석(ANOVA)을 통해 이를 검증하였다.

---

## 🎯 연구 배경 및 필요성
- 최근 고물가 및 소비 위축 환경에서 **지역별 소비 패턴의 구조적 차이**에 대한 이해 필요
- 기존 상권 분석은 단순 매출 규모 중심 → **소비 구조 관점의 분석 필요**
- 서울시 내 권역별 소비 특성을 파악하여 **지역 맞춤형 전략 수립 가능성 탐색**

---

## 🔍 연구 목적
1. 서울시 5대 권역 간 소비 항목 비중 차이가 존재하는지 검정
2. 차이가 존재할 경우, **어떤 소비 항목이 특정 권역에서 상대적으로 높은지** 확인
3. 소비 비중 기반 분석을 통해 **지역 간 비교 가능성 확보**

---

## 🗂 데이터 설명
- **데이터 출처**: 서울 열린데이터 광장  
- **데이터명**: 서울시 상권분석서비스 (소득·소비·자치구)
- **분석 단위**: 행정동 → 자치구 → 권역 단위로 재구성

## Raw Data
- Source: Seoul Open Data Plaza
- Dataset: Commercial Analysis Service (Income & Consumption, District-level)
- Encoding: EUC-KR
- File: seoul_income_consumption_gu.csv

### 주요 변수
- 총지출
- 교육 지출
- 여가 지출
- 의료 지출
- 교통 지출

---

## ⚙️ 데이터 전처리
- 변수명 표준화 및 자료형 변환
- 결측치 확인 (결측치 없음)
- 행정동 코드를 활용하여 자치구 단위 재구성
- 서울시 공식 기준에 따라 **5대 권역(도심, 동북, 서북, 서남, 동남)**으로 그룹화

---

## 🧠 파생 변수 생성
지역 간 비교를 위해 절대 금액이 아닌  
**총지출 대비 소비 항목 비중 변수**를 생성하였다.

```python
교육비중 = 교육지출 / 총지출
여가비중 = 여가지출 / 총지출
의료비중 = 의료지출 / 총지출
교통비중 = 교통지출 / 총지출
```
👉 이를 통해 자치구별 총지출 규모 차이를 보정하였다.

- 중요 정보 손실 가능성을 고려하여 **이상치는 제거하지 않음**

## 📊 데이터 시각화
- **Box Plot**: 권역별 소비 비중 분포 비교
- **Violin Plot**: 분포 형태 및 밀도 확인  
- 시각화를 통해 일부 소비 항목에서 **권역 간 분포 차이 관찰**

---

## 📐 통계 분석 (ANOVA)

### 가설 설정
- **H₀ (귀무가설)**: 서울시 권역 간 소비 항목 비중의 평균 차이는 존재하지 않는다.
- **H₁ (대립가설)**: 서울시 권역 간 소비 항목 비중의 평균 차이는 존재한다.

### 분석 방법
- 일원 분산분석 (**One-way ANOVA**) 수행
- 소비 항목별로 권역 간 평균 차이 검정

---

## 📌 분석 결과 요약
- 일부 소비 항목에서 **권역 간 소비 비중 차이가 통계적으로 유의미함**
- 특정 권역에서 **교육·여가·교통 소비 비중이 상대적으로 높게 나타남**
- 이를 통해 **권역별 소비 구조가 상이함**을 확인
